# ai-security-threat

ai-security-threat SKILL知识库

## 简介

AI/LLM应用安全威胁评估工具。用于安全审查、威胁分析、生成安全评估报告。

## 威胁知识库

本知识库包含**26个详细的威胁文档**，位于 `threats/` 目录下，按照AI应用的分层架构组织：

### 1. 提示词层安全 (Prompt Layer)
- 系统提示词涉及敏感信息
- 提示词泄露与知识产权
- 单模态：文字交互提示词注入
- 多模态：间接提示词注入
- 多模态：ASCII走私

### 2. 模型层安全 (Model Layer)
- 模型越狱：混淆编码、角色扮演
- 模型拒绝服务：模型长上下文控制缺失、模型的并发控制缺失、模型交互资源耗尽
- 模型逻辑缺陷：渐进式攻击威胁、训练数据交互式获取、模型恶意代码内容生成、模型有害内容生成、模型幻觉与事实性错误
- 模型操作执行：系统命令代码执行或其他恶意操作注入、恶意偏见决策执行
- 模型特有攻击：对抗样本攻击、模型供应链攻击、模型逆向与隐私泄露

### 3. 模型训练安全 (Training Security)
- 训练数据知识库数据范围控制缺失
- 训练数据投毒

### 4. 智能体层安全 (Agent Layer)
- MCP协议安全：MCP权限校验缺陷、混淆副手缺陷、令牌透传、会话劫持
- 权限控制：封装层API越权-IDOR、封装层服务未授权、封装层服务功能级别的越权
- 传统Web安全：SQL注入、XSS跨站脚本、前端敏感信息泄露、任意文件上传、任意文件下载、任意文件删除、路径遍历
- 日志审计：日志缺失风险

## 使用方式

### 查询特定威胁
查看 `threats/` 目录下的威胁文档，每个文档包含：
- 漏洞原理
- 漏洞危害
- 修复方式

### 理解威胁关联
先阅读 `威胁关联分析.md`，了解威胁之间的关联关系、典型攻击链和防御策略。

### 生成安全评估报告
使用 `templates/security-report-template.md` 模板生成专业的安全评估报告。
